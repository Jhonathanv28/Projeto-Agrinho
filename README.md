# 🌱 Projeto Agrinho 2026

## Agro forte, futuro sustentável

Projeto desenvolvido para o concurso **Agrinho 2026**, com o tema **"Agro forte, futuro sustentável"**, destacando a importância do equilíbrio entre produção agrícola, preservação ambiental e desenvolvimento social.

O site apresenta informações educativas sobre os desafios do campo, soluções sustentáveis e os impactos positivos da adoção de boas práticas agrícolas, além de recursos interativos para engajar os visitantes.

## 🚀 Funcionalidades

- 📖 Apresentação do tema sustentabilidade no agro;
- 🌾 Explicação dos principais desafios enfrentados pelo campo;
- 💡 Demonstração de soluções sustentáveis;
- 📊 Exibição dos benefícios econômicos, ambientais e sociais;
- ♻️ Simulador de atitudes sustentáveis;
- ❓ Quiz educativo sobre sustentabilidade;
- 📱 Layout responsivo para diferentes dispositivos.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Conscientizar sobre a importância de práticas sustentáveis no agronegócio, mostrando que é possível aumentar a produção de alimentos enquanto se preservam os recursos naturais para as futuras gerações.

## 🌍 Temas Abordados

- Produção agrícola sustentável
- Conservação dos recursos naturais
- Mudanças climáticas
- Uso consciente da água
- Preservação da biodiversidade
- Energia renovável
- Educação ambiental
